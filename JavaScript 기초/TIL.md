# 📘 Today I Learned

### 1. 오늘 배운 내용
  1) ID의 중요성: </br>
     자바스크립트가 HTML 입력창을 찾지 못해 데이터가 들어오지 않는 문제 발생함.</br>
     각 input, textarea, select 태그에 id를 부여하여 자바스크립트가 getElementById로 정확하게 데이터 타겟팅 가능
  2) span태그 사용</br>
     입력창 아래에 빨간 색 경고 문구를 넣는데에 활용
  3) isValid라는 깃발 변수를 활용하여 모든 조건이 충족될 때에만 카드가 추가되도록 구현

### 2. 핵심 정리 (내 언어로)
html에 id를 부여하여 자바스크립트가 올바른 타겟을 읽도록 하고, 유효성 검사에서 isValid를 통해 조건이 충족되는지 확인한 후 카드를 생성하도록 하였다.

### 4. 느낀 점
이번에는 자바스크립트의 id와 html을 연결하는데에 큰 어려움을 겪었다. 정확한 이름과 타겟을 지정하는 구문을 작성하는 것이 얼마나 큰 영향을 끼치는지 배울 수 있었다.</br>
또 목록을 추가하는 폼 페이지가 열리긴 해도, 타겟 지정이 잘못되어 프로그램이 받을 데이터가 null, 즉 없다고 판단하여 오류를 범한 곳이 많았다. </br>
이를 통해 id지정하기에 명확한 이름 짓기가 큰 역할을 한다는 것을 느꼈다.
