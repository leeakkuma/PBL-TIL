# 📘 Today I Learned

### 1. 오늘 배운 내용
1. css 파일이 따로 존재했다!
   .css와 .html 파일을 따로 만들 수 있다.
   덕분에 html부분과 style부분이 구분되어 길고 긴 코드를 덜 복잡하게 봐도 되었다.
2. grid layout을 설정하여 1개 이상의 카드들을 정렬할 수 있다
   만들어둔 카드들을 card-wrapper 클래스로 묶어두고 행과 열을 설정하여 그리드를 만들 수 있었다.
3. 화면 크기에 따라 정렬되는 카드들의 개수를 정할 수 있다.
   화면 크기가 768px, 1024px일 때 1행을 3번 놓는다는 명령문을 사용하여 열을 쌓고 그리드를 만든다.
### 2. 핵심 정리 (내 언어로)
display를 grid로 설정하여 grid layout을 사용함을 설정한다. 이런 gird를 사용하면 복잡하고 긴 코드를 간단하게 나타낼 수 있다.
